This project implements a 5-sensor line following robot using Arduino Nano.

## Hardware
- Arduino Nano
- DC-DC Buck 5V 3A Dual USB Charge Module
- TCRT5000 Infrared Reflective Sensor
- Switch
- Pin 9V
- GA12 N20 V2 43mm Motor Wheel
- CY15A ball bearing universal wheel
- IR Infrared Obstacle Avoidance
- 9V battery connector
- Chasiss Baby Cat Mecanum Robot Chassis
- L298N motor driver
- 5 IR line sensors
- 2 DC motors

## Pin Configuration
See `config.h`.

## Usage
1. Open `LineFollowerBot.ino` in Arduino IDE
2. Select board: **Arduino Nano**
3. Upload and test!
